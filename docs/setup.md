# Project Setup

## Requirements
- Python 3.9+
- pip

## Setup Instructions

### 1. Clone the repository
```bash
git clone <repo-url>
cd connected-dots
```

### 2. Create Virtual Environment (optional)
```bash
python -m venv venv
```
### 3. To Activate Virtual Environment
#### Windows
```bash
venv\Scripts\activate
```
#### Linux/macOS
```bash
source venv/bin/activate
```

### 4. Install Dependencies
```bash
pip install -r requirements.txt
```

### 5. Run test
```bash
run cmd
```
